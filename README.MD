# Optimización Semidefinida

Este repositorio está dedicado a la materia universitaria de Optimización Semidefinida.

La programación semidefinida (SDP) es una subdisciplina de la optimización convexa que se ocupa de la optimización de una función lineal sujeta a restricciones de que una matriz de variables sea semidefinida positiva. Es una poderosa herramienta con aplicaciones en diversas áreas como la ingeniería de control, la combinatoria, la teoría de grafos y el aprendizaje automático.

Aquí encontrarás ejercicios y ejemplos relacionados con los temas cubiertos en la materia de Optimización Semidefinida.
